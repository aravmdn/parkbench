Diagnostic profile viewer

Reading an agent's diagnostic profile

This viewer renders the three diagnostic outputs Parkbench produces — the radar (capability across four axes), the career ("park tour" where reputation compounds), and the leaderboard (ranked careers). Use Open JSON to load a radar --json, career --json, or leaderboard --json payload; the kind is auto-detected.

Note: some browsers block fetch() on file:// URLs, so the bundled Load samples may not work when this page is opened as a plain file. If that happens, serve the folder with a local server (python -m http.server 8080 --directory viewer/) or use Open JSON to pick a file from disk — that always works. You can also pass ?path=relative/to/file.json (same origin).
